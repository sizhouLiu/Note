还有append这样的函数操作 使得这一列表非常容易操作

另一个就是字典

还记得适应期的字典的折磨（字典还好 超市老板是真折磨)

但是如果有Python的字典 相信会非常简洁的就实现所有功能

第三是生成器

这也是C语言没有的（大概吧 也许有办法可以实现类似功能 我不知道）
